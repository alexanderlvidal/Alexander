Making a website

Below you will find a description of my process making this website.

Objective

The objective of this project was to create a functional website using HTML5 and write in detail how the website was created. The website should also have my contact information as well as my resume.

Methodology

Step 1

I learned the basics from lectures and the website W3Schools.com provited by my professor.

Step 2

I made an account at github.com and downloaded the programs: brackets og git-scm.

Step 3

I found a free open source template at html5up and I uploaded the template to the Brackets application

Step 4

Finally I began to edit and write code in the template to display what I wanted until I was happy with the results.

Step 5

On the github website I created a new repository and uploaded all my files and published the website.

Then my website was completed :)
